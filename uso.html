Como utilizar o simulador do protocolo FTP

Instruções de uso

  1. Para utilizar o simulador, primeiro ligue o servidor e informe o IP do servidor IP: 172.16.62.36.
  2. Como dados de usuário use admin como nome de usuário e admin como senha.
  3. Crie um arquivo ou transfira um arquivo existente degitenado o nome do arquivo e cliqaundo no botão enviar.
  4. Para baixar um arquivo digite o nome do arquivo e clique no botão baixar.

Simulações implementadas

  • Conectar com um servidor remoto
  • Validação do usuário pelo servidor
  • Subir arquivos ao servidor
  • Baixar arquivos do servidor
  • Se desligar o servidor durante uma transferência de arquivos a transferência e abortada./li>
